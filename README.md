# 🐣 Mi Tamagotchi

¡Bienvenido a mi Tamagotchi! Un jueguito creado por mí para que cuides de tu mascota virtual.

## 🎮 ¿Cómo funciona?
* Alimenta a tu mascota cuando tenga hambre.
* Juega con ella para que no se aburra.
* ¡Limpia sus necesidades!
* **Objetivo:** Mantenerla viva la mayor cantidad de días posible.
